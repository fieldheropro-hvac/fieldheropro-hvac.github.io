GITHUB PAGES — PHONE-FRIENDLY DEPLOYMENT
1. Sign in to GitHub and create a new PUBLIC repository named:
   fieldhero-decision-desk
2. Open the repository and choose Add file > Upload files.
3. Upload all five files from this package. Do not upload the ZIP itself.
4. Commit the files.
5. Open Settings > Pages.
6. Under Build and deployment choose:
   Source: Deploy from a branch
   Branch: main
   Folder: /(root)
7. Save.
8. GitHub will show the public HTTPS address after deployment.
